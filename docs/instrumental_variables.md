# Instrumental Variables (IV)

## Problem
Endogeneity due to omitted variables or reverse causality.

## Solution
Use an instrument Z such that:
- Z correlated with X
- Z independent of error term

## Estimation
- Two‑Stage Least Squares (2SLS)

## Diagnostics
- Weak instrument tests
- Overidentification tests

## Warning
Bad instruments make results worse than OLS
