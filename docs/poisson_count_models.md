# Poisson and Count Models

## Purpose
Models count data where Y ≥ 0.

## Assumptions
- Event counts are independent
- Mean equals variance

## Red Flags
- Excess zeros
- Overdispersion

## Alternatives
- Negative Binomial
- Zero‑inflated models

## Interpretation
- Coefficients interpreted as log changes in expected counts

## Diagnostics
- Residual plots
- Mean‑variance comparison
