# Logit and Probit Models

## Purpose
Used when the dependent variable is binary.

## Model Types
- Logit: Logistic distribution
- Probit: Normal CDF

## Model Form
P(Y=1 | X) = F(Xβ)

## Key Characteristics
- Nonlinear
- Estimated via maximum likelihood
- Coefficients not directly interpretable

## Interpretation
- Use marginal effects
- Odds ratios (logit only)

## Model Evaluation
- Confusion matrix
- Accuracy
- ROC / AUC

## Limitations
- Hard to interpret interactions
- Sensitive to class imbalance
